ℹ️ Help & Information

How to Use

1. Load NFTs

  • Paste wallet addresses in the sidebar (one per line)
  • Click "Load NFTs into Database" to fetch and store your collection

2. Analyze Synergies

  • Select a skin tone, attribute group, and optionally a wallet filter
  • View synergies grouped by attribute value and rows
  • 🔥 indicates synergies that meet the threshold
  • Stickers are selected to maximize diversity and value

3. Database

  • Data is stored in SQLite after loading
  • Subsequent loads update existing stickers by address
  • Use "Clear Database" to start fresh

4. Exchange Basket

  • The 🛒 Exchange Basket tab contains two sub‑sections:
    • Исходные кошельки – показывает, из каких кошельков вы добавили стикеры.
    • Распределение обмена – здесь располагаются сами корзины для обмена: один пул + по одной корзине на каждый кошелёк.
  • Добавляйте стикеры в корзину, они попадут в пул и автоматически исключатся из портфолио.
  • В разделе распределения перетаскивайте карточки между корзинами, чтобы моделировать обмены; портфолио будет пересчитано по выбранному цвету, а рядом с каждой корзиной отобразится разница в PWR.
  • Во вкладке «Новый портфель» показывается итоговое состояние каждого кошелька после обмена: команды, силы и изменения по сравнению с исходным портфелем.

5. Sticker Values

The scoring system is based on:

  • Mythic+: 1000 points
  • Mythic: 500 points
  • Legendary: 200 points
  • Epic: 140 points
  • Rare: 80 points

Plus additional bonuses from synergies and name patterns.
